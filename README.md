# Dromerij | Eklectische Aanval

This repository contains the source code that generates a static website for my pottery and arty-farty atelier. 
Name is in flux but something like Dromerij ('the dreamery' in Dutch) or Eklectische Aanval ('Eclectic Attack', a play on words on Epileptic).

This is hosted on github pages [here](https://ppdelft.github.io/ppdelft/).
Rebuilt on push to main. 

## 🧞 Dev Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `just setup`              | Installs dependencies                            |
| `just dev`                | Starts local dev server at `localhost:4321`      |
| `just build`              | Build your production site to `./dist/`          |
| `just serve`              | Preview your build locally, before deploying     |
| `just test`               | Run all tests                                    |
| `just add-gallery`        | Add image to the gallery                         |
